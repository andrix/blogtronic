Blogtronic
==========

Simple blog system for lazy people that runs on Google App Engine.
Disclaimer: Udacity assignament


How to hack?
------------

Clone it from Github

    git clone git@github.com:andrix/blogtronic.git

Unzip the file

    unzip blogtronic.zip


How to run using Google App Engine dev server?
----------------------------------------------

Go to the project folder, and run:

    dev_appserver.py .

Then go to localhost:8080 (if your port is other, just change it to the one you use)


Deploy to Google App Engine
---------------------------

* Guide to deploy: https://cloud.google.com/appengine/docs/python/tools/uploadinganapp
